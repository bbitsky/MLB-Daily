## ⚠️ Script Error — DB Path Issue

`mlb_results.py` and `mlb_dashboard.py` both failed with:
```
sqlite3.OperationalError: unable to open database file
```

**Root cause:** The project folder name "MLB Daily" contains a space. The Linux sandbox's sqlite3 cannot open the DB through a FUSE-mounted path with spaces. This works fine on your Windows machine but breaks in the automated environment.

**Workaround:** Results below were fetched manually via web search (ESPN/CBS Sports) and computed directly.

**Fix needed:** Rename project folder to `MLB_Daily` (no space), or update `DB_PATH` in `mlb_data.py` to use a path without spaces.
